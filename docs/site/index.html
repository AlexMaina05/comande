Caratteristiche Principali

  • Gestione Ordini: Crea ordini con ricerca prodotti in tempo reale
  • Ordini Staff: Traccia i pasti dello staff separatamente (costo zero, esclusi dai report)
  • Supporto Sconti: Applica sconti agli ordini dei clienti
  • Stampa per Reparto: Generazione automatica di ricevute per cucina, bar e altri reparti
  • Report Giornalieri: Statistiche di vendita e ricavi con export PDF
  • Pannello Admin: Gestisci prodotti, tavoli, categorie e reparti

Stack Tecnologico

  • Frontend: HTML, CSS, JavaScript
  • Backend: PHP 8.0+
  • Database: MySQL/MariaDB
  • Stampa: Integrazione CUPS per stampa ricevute
  • API: Endpoint REST JSON
